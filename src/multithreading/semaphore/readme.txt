---///Semaphore///---
- это синхронизатор, позволяющий ограничить доступ к какому-то ресурсу.
В конструктор Semaphore нужно передавать количество потоков, которым Semaphore будет разрешать одновременно использовать
этот ресурс.

acquire() - метод предназначен для попытки получить разрешение у семафора.
