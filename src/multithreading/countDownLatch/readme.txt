---///CountDownLatch///---
CountDownLatch - это синхронизатор, позволяющий любому количеству потоков ждать,
пока не завершится определённое количество операций. В конструктор CountDownLatch нужно передавать количество операций,
которые должны завершиться, чтобы потоки продолжили свою работу.
