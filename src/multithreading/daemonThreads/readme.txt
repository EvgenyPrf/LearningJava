---///Daemon threads///---
Daemon треды предназначены для выполнения фоновых задач и оказания различных сервисов User тредам.
При завершении работы последнего User треда программа завершает своё выполнение, не дожидаясь окончания работы демон-треда.
Например: большинство потоков JVM являются демон-тредами. Они занимаются сборкой мусора (Garbage collector), освоббождают память, удаляя ненужные объекты.
Чтобы сделать юзер-тред демон-тредом, необходимо перед запуском треда, вызвать у него метод setDaemon(true).
Чтобы проверить, является ли тред демоном, нужно вызвать у него метод isDaemon().
